loop-eval: A Bidirectional Quality Substrate

Where it fits today, the net-new it unlocks, and the Rubrica first instance

June 25, 2026

The unlock. loop-eval is not a regression CI. It is a single quality substrate with a knob that turns both ways. The same frozen ground truth, objective, and regime power two directions: defend (block any change that scores below baseline) and improve (hill-climb to find a candidate that beats baseline, then promote it). The pivot is that the same baseline is both the floor you cannot drop below and the bar to beat. Author the labeled set once; get both directions for free. Quality stops being a hope and becomes a ratchet — never down, systematically up.

The two knobs

Both run off one project instance (frozen ground truth + a pluggable objective + regime versioning). Only the entry point and the intent differ.

DirectionEntry pointWhat it doesWhen it fires
Defend (regression CI) loop_eval.verify Scores a candidate prompt/model against the regime baseline on the frozen set; exits non-zero if it regresses or trips an anomaly/overfit flag. Every prompt or model change, in CI — a guardrail before it ships.
Improve (search) loop_eval.loop Hill-climbs the prompt against the objective, escalates the winner through the holdout overfit gate; promotes only if it beats baseline and generalizes. On demand, when you want a better prompt — actively turns the knob up.
The CI/CD analogy makes it concrete: regression CI is "tests must pass." The improvement loop adds "...and here is a candidate that scores higher — promote it." Same harness, same ground truth, opposite intent. The regime stamp guarantees the gate and the search are always comparing on identical terms.

First instance: projects/rubrica-grading/

Start here. The fit is exact, the ground truth already exists, and it converts loop-eval from "a thing I built" into a guardrail that protects the product Travis actually ships grades from — while also being the search loop that improves it.

The adapters (the only edit surface)

FileResponsibility
ingest.pyPull (AI first-pass, human-final) per question from the calibration CSVs / finalexam DB — reuse the reconstruct_final.py logic. Ground truth = Travis's released grades (the rule he already holds: baseline only on ground truth you authored).
runner.pyCall Rubrica's grading prompt on one question's answer, return the numeric score. Arithmetic (sum/clamp) in code; the model emits per-part judgments only.
mutations.pyStructural grading-instruction operators (criteria, leniency, carry-forward) — biased toward structure, matched to the grader's failure direction per the loop-eng lesson.
constraints.jsonlWhere Claude/human grading verdicts re-enter as two-sided policy, with the consolidation review guarding against rot.
config.yamlLevers: salt, holdout %, the frozen exam set, the objective (within_tol, climb: within|mae per question), model endpoint.

Both knobs, day one

Defend: python -m loop_eval.verify --project projects/rubrica-grading --predictions <new-prompt-output> --split gap — exits 2 if a prompt edit regresses below the released-grade baseline or overfits. Wire into the Rubrica repo's pre-merge check.

Improve: python -m loop_eval.loop --project projects/rubrica-grading --escalate — hill-climbs the grading prompt, escalates the winner on the holdout, promotes only if it beats the baseline and generalizes.

Boundary to hold. The grading logic stays human-driven (per agent-task-design): the objective definition and the ground truth are Travis's levers, never the loop's. loop-eval optimizes whatever the objective measures, flaws included — so the human owns the objective, and the loop owns the search.

Recommended path

Build projects/rubrica-grading/ as the first bidirectional instance. It proves both knobs on a product with real stakes and pre-existing ground truth. Once it's live, the same template clones to SiteProof pricing (extraction objective) and the cosmos VLM bench (judge objective), and the net-new prompt-quality ratchet generalizes to every production LLM feature. The substrate built once becomes the quality floor and ceiling for everything Travis ships.
